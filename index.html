CSS Selectors and their Specificity

Selector Description Name Description Specificity Value
#IDname ID The #IDname selects a single element to give it its own unique styling. 100
.classname Class The class attribute selects all HTML elements that contain this class attribute. The most common way to select an element. 10
[attribute] Attribute The attribute selector can be used to target all HTML elements that contain that attribute. 10
p, h1, h2, etc. Type The type selector matches the type of the element in HTML without the angle brackets. 1
* Universal The Universal selector selects all elements of any type. It's used for resetting default browser styling and for selecting all children of a parent element. 0
